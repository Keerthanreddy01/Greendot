# Security Policy

## Supported Versions
We currently support security fixes on the `main` branch.

## Reporting a Vulnerability
Please report security issues privately to:

- Email: `keerthanreddy1706@gmail.com`
- Do not open public GitHub issues for vulnerabilities

We will acknowledge your report within 72 hours and provide a timeline for remediation.

## Guidelines
- Provide steps to reproduce and affected components
- Include environment details (Flutter/Dart versions, platform)
- If possible, include a minimal proof-of-concept

Thank you for helping keep GreenDot and its users safe.
